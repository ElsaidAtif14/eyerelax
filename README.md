# 👁️ EyeRelax - VS Code Extension

**EyeRelax** is a productivity tool designed for developers to prevent eye strain by following the **20-20-20 rule**. 

## ✨ Features
* 🕒 **Automatic Reminders:** Get notified every 20 minutes to take a break.
* 💤 **Status Bar Integration:** A sleek icon and timer in your VS Code status bar.
* 🚀 **Quick Action:** Click the status bar to trigger a break manually.
* 🎨 **Beautiful UI:** Uses a custom sleeping-mask icon for a friendly look.

## 🛠️ How to Use
1. Install the extension from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=elsaid-atif.eyerelax).
2. Look at the bottom right of your VS Code window.
3. Every 20 minutes, a notification will pop up.
